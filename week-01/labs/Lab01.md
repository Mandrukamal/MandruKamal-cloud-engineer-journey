### Lab 1 — S3 Hands-On (45 mins)

Go to your AWS Console and do this:
## Task 1 — Create a bucket:

Bucket name: kamal-cloud-journey-[yourname]
Region: ap-south-1 (Mumbai — closest to you)
Block all public access: ON for now
Versioning: Enable it

## Task 2 — Upload objects:

Upload any 2 files (text files are fine)
Explore the object URL
Check storage class — should be Standard

## Task 3 — Lifecycle rule:

Create a lifecycle rule
Transition objects to S3-IA after 30 days
This is a real cost optimization pattern used in production
